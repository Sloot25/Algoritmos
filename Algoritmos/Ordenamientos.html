BubbleSort

Algoritmos de Ordenamiento

View the project on GitHub

Resumen

El algoritmo de BubbleSort es un algoritmo de ordenamiento, conocido por ser uno de lo más ineficientes que puede existir, tomando parte de una complejidad cuadrática, el algoritmo consiste en el posicionamiento de cada elemento en la posición en la que debería colocarse en orden creciente.

El problema de BubbleSort radica en su idea centrar, recorremos nuestros elementos, que generalmente los vemos en forma de Arreglo o Lista. Es de esta forma que compararemos el elemento actual con el siguiente, si este es mayor que el siguiente entonces los intercambiamos de lugar. Repetiremos este proceso hasta haber recorrido cada uno de los elementos.

A continuación tenemos el ejemplo de una implementación de BubbleSort: 


public class BubbleSort {
    public static void main(String[] args) {
        int[] arr = {64, 34, 25, 12, 22, 11, 90};
        bubbleSort(arr);
        System.out.println("Sorted array");
        printArray(arr);
    }

    static void bubbleSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n-1; i++) {
            for (int j = 0; j < n-i-1; j++) {
                if (arr[j] > arr[j+1]) {
                    // swap arr[j] and arr[j+1]
                    int temp = arr[j];
                    arr[j] = arr[j+1];
                    arr[j+1] = temp;
                }
            }
        }
    }

    static void printArray(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n; ++i)
            System.out.print(arr[i] + " ");
        System.out.println();
    }
}

SelectionSort

Algoritmos de Ordenamiento

View the project on GitHub

Resumen

El algoritmo de SelectionSort es un algoritmo de ordenamiento, conocido por ser uno de lo más ineficientes que puede existir, tomando parte de una complejidad cuadrática, si bien el algoritmo es cuadratico al igual que BubbleSort, Selectionsort es un algoritmo más rapido que BubbleSort

SelectionSort no consiste en una comparación a pares durante n veces para ordenar los elementos, la idea central de selectionSort consiste en ordenar cada uno de los elementos en orden creciente, es decir, en primer instancia ordenaremos los menos y posicionandolos en su respectivo orden, haciendo que busquemos el minimo de los elementos restantes y lo coloquemos en la posición actual.

A continuación tenemos el ejemplo de una implementación de SelectionSort: 

  
    public class SelectionSort {
    public static void selectionSort(int[] arr) {
        int n = arr.length;

        for (int i = 0; i < n - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }

            // Intercambiar el elemento mínimo con el primer elemento
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }

    public static void main(String[] args) {
        int[] arr = {64, 25, 12, 22, 11};
        System.out.println("Array original:");
        printArray(arr);

        selectionSort(arr);

        System.out.println("Array ordenado:");
        printArray(arr);
    }

    public static void printArray(int[] arr) {
        for (int i : arr) {
            System.out.print(i + " ");
        }
        System.out.println();
    }
}

QuickSort

Algoritmos de Ordenamiento

View the project on GitHub

Resumen

El algoritmo de QuickSort es un algoritmo de ordenamiento, conocido por todos, como podemos imaginar por su nombre, es un algoritmo rapido, si bien su complejidad sigue siendo cuadratica en el peor de los casos, en el caso promedio tenemos una complejidad dada por O(n log n)

QuickSort es un algoritmo que depende de la suerte, el algoritmo consiste en tomar un pivote, a partir del cual separaremos a los elementos en los menores al pivote y los mayores al pivote. Al haber terminado eso es que realizaremos recursión en los elementos a la izquierda y a la derecha hasta que lleguemos a un unico elemento.

Es por ello que podemos notar donde esta nuestra aleatoriedad, y esto se encuentra en la toma de nuestro pivote. Por ello es que en el peor de los casos tendremos un selectionSort.

A continuación tenemos el ejemplo de una implementación de QuickSort: 

  
    public class QuickSort {
    public static void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            int pi = partition(arr, low, high);

            quickSort(arr, low, pi - 1);
            quickSort(arr, pi + 1, high);
        }
    }

    private static int partition(int[] arr, int low, int high) {
        int pivot = arr[high];
        int i = (low - 1);
        for (int j = low; j < high; j++) {
            if (arr[j] < pivot) {
                i++;
                int temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }
        }

        int temp = arr[i + 1];
        arr[i + 1] = arr[high];
        arr[high] = temp;

        return i + 1;
    }

    public static void main(String[] args) {
        int[] arr = {10, 7, 8, 9, 1, 5};
        int n = arr.length;

        System.out.println("Array original:");
        printArray(arr);

        quickSort(arr, 0, n - 1);

        System.out.println("Array ordenado:");
        printArray(arr);
    }

    public static void printArray(int[] arr) {
        for (int i : arr) {
            System.out.print(i + " ");
        }
        System.out.println();
    }
}

MergeSort

Algoritmos de Ordenamiento

View the project on GitHub

Resumen

El algoritmo MergeSort es un algoritmo de ordenamiento que nos toma tiempo n log n, si bien en el caso promedio nos toma más tiempo que QuickSort, en el peor caso nos toma menos tiempo, siendo de esta manera que el algoritmo tiene complejidad logaritmica.

MergeSort consiste en la división de nuestros elementos en dos partes, las cuales van a ser divididas y se les aplicará recursión hasta que solo nos queden listas de un unico elemento. Es en este momento donde nos encargararemos de unir las dos listas resultantes de tal manera que la unión de estas sea ordenada, sabiendo que unir dos listas ordenadas nos toma tiempo lineal, es facil ver que la complejidad de este algoritmo está dada por n log n.

A continuación tenemos el ejemplo de una implementación de MergeSort: 

  
   public class MergeSort {

    public static void mergeSort(int[] arr, int left, int right) {
        if (left < right) {
            int mid = (left + right) / 2;

            // Ordenar la primera y segunda mitad
            mergeSort(arr, left, mid);
            mergeSort(arr, mid + 1, right);

            // Fusionar las mitades ordenadas
            merge(arr, left, mid, right);
        }
    }

    private static void merge(int[] arr, int left, int mid, int right) {
        int n1 = mid - left + 1;
        int n2 = right - mid;

        // Crear arreglos temporales
        int[] L = new int[n1];
        int[] R = new int[n2];

        // Copiar datos a los arreglos temporales
        for (int i = 0; i < n1; ++i)
            L[i] = arr[left + i];
        for (int j = 0; j < n2; ++j)
            R[j] = arr[mid + 1 + j];

        // Fusionar los arreglos temporales

        int i = 0, j = 0;

        // Índice inicial del subarreglo fusionado
        int k = left;
        while (i < n1 && j < n2) {
            if (L[i] <= R[j]) {
                arr[k] = L[i];
                i++;
            } else {
                arr[k] = R[j];
                j++;
            }
            k++;
        }

        // Copiar los elementos restantes de L[], si hay alguno
        while (i < n1) {
            arr[k] = L[i];
            i++;
            k++;
        }

        // Copiar los elementos restantes de R[], si hay alguno
        while (j < n2) {
            arr[k] = R[j];
            j++;
            k++;
        }
    }

    public static void main(String[] args) {
        int[] arr = {12, 11, 13, 5, 6, 7};
        System.out.println("Array original:");
        printArray(arr);

        mergeSort(arr, 0, arr.length - 1);

        System.out.println("Array ordenado:");
        printArray(arr);
    }

    public static void printArray(int[] arr) {
        for (int i : arr) {
            System.out.print(i + " ");
        }
        System.out.println();
    }
}

CountingSort

Algoritmos de Ordenamiento

View the project on GitHub

Resumen

A diferencia de los otros algoritmos basados en comparaciones, este algoritmo usa condiciones previas para poder asegurar un ordenamiento en tiempo lineal. Estas condiciones previas consisten en saber la cota superior de los elementos y estos puedan compararse con enteros.

CountingSort consiste en que al saber el numero maximo que se puede encontrar en nuestros elementos, entonces crearemos un arreglo desde 0 hasta el maximo y recorreremos cada uno de los elementos posicionandolos en la casilla correspondiente a su representación entera. Posteriormente recorreremos este arreglo recuperando los elementos

A continuación tenemos el ejemplo de una implementación de BucketSort: 

  
    public class CountingSort {
    public static void countingSort(int[] arr) {
        int n = arr.length;

        if (n == 0) {
            return;
        }

        // Encuentra el valor máximo en el array
        int max = arr[0];
        for (int i = 1; i < n; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
        }

        // Inicializa el array de conteo
        int[] count = new int[max + 1];

        // Almacena el conteo de cada valor
        for (int i = 0; i < n; i++) {
            count[arr[i]]++;
        }

        // Modifica el array de conteo
        for (int i = 1; i <= max; i++) {
            count[i] += count[i - 1];
        }

        // Construye el array ordenado
        int[] output = new int[n];
        for (int i = n - 1; i >= 0; i--) {
            output[count[arr[i]] - 1] = arr[i];
            count[arr[i]]--;
        }

        // Copia el array ordenado al array original
        for (int i = 0; i < n; i++) {
            arr[i] = output[i];
        }
    }

    public static void main(String[] args) {
        int[] arr = {4, 2, 2, 8, 3, 3, 1};
        System.out.println("Array original:");
        printArray(arr);

        countingSort(arr);

        System.out.println("Array ordenado:");
        printArray(arr);
    }

    public static void printArray(int[] arr) {
        for (int i : arr) {
            System.out.print(i + " ");
        }
        System.out.println();
    }
}
